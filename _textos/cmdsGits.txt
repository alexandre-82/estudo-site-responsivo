Lista de comandos para atualizar o projeto no github

1 passo:

sudo git add .

2 passo:

sudo git commit -m 'DESCRICAO DO COMMIT'

3 passo:

sudo git push --all

criar um branch
git checkout -b nome_novo_branch

mover para um branch criado
git checkout nome_branch


passar as alterações do branch html para o master:

sudo git add .
sudo git commit -m 'DESCRICAO DO COMMIT'
sudo git push --all
git checkout master

git merge html
